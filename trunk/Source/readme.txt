bbv.DomainDrivenDesign
----------------------

NOTE: To use this library you must install .NET 3.5 Service Pack 1 and 
Visual Studio 2008 Service Pack 1:
http://www.microsoft.com/downloads/details.aspx?familyid=FBEE1648-7106-44A7-9649-6D9F6D58056E&displaylang=en

Make sure you have installed SQL Management Studio Express:
http://www.microsoft.com/downloads/details.aspx?FamilyID=c243a5ae-4bd1-4e3d-94b8-5a0f62bf7796&displaylang=en

This library is an implementation of DDD and Enterprise Application Patterns using Linq to Entities and Linq to Sql.
For more information see the include documentation.


To build the API-Documentation with Sandcastle you must install
Sandcastle Version 2.4.10520.1: http://www.microsoft.com/downloads/details.aspx?FamilyId=E82EA71D-DA89-42EE-A715-696E3A4873B2&displaylang=en
Sandcastle Help File Builder Version 1.7.0.0: http://shfb.codeplex.com/Release/ProjectReleases.aspx?ReleaseId=10445
